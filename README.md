   _Requires [Docker](https://www.docker.com/) installed._
   ```bash
   docker-compose up -d
   ```

6. **Run the app**
   ```bash
   npm run dev
   ```

7. **Open the app**  
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
